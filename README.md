## 📕 SOBRE O PROJETO

Projeto de implementação de uma Loja ecommerce de plantas utilizando React..

## 📌 OBJETIVOS

O Adminstrador deve ser capaz de visualizar sua pagina, adicionar novas plantas e ter a opção de aplicar descontos ou não.

  - versão 18.2.0

- JavaScript
  - Versão 7.0.2.2

- Node
  - Versão 18.16
